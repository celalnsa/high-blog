函数式编程实践-缓存函数执行结果

发表于 2017-04-27 | 分类于 术业 | 阅读次数

受函数式编程思想启发, 实现Python函数缓存装饰器。

#纯函数

学习函数式编程可以看JS函数式编程指南中文版
纯函数可测，可重演，最重要的是，因为同样的输入一定会产生同样的输出，使得函数的执行结果是可以被缓存的。

expired是函数缓存的有效时间。
可以看到在new_func的逻辑当中，只有当缓存里没有这个key，或者缓存的时间已经过期时，才会重新执行原来的函数func并更新缓存。
